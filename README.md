# ATM Management System

A console-based ATM Management System developed in Java that simulates real-world ATM operations.

## Features
- Card Insertion & PIN Authentication
- Balance Inquiry
- Cash Deposit
- Cash Withdrawal
- Fast Cash Withdrawal
- Mini Statement Generation
- PIN Change Functionality
- Transaction History with Timestamps

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework
- Date-Time API
- Git & GitHub

## Project Structure
- Main.java – ATM workflow and menu operations
- Account.java – Account details and transaction handling
- Transaction.java – Transaction logging with timestamps

## Future Enhancements
- File Handling for persistent storage
- Multiple account support
- Admin dashboard
- Database integration using MySQL
